Introduction

This document summarizes the technical capabilities of this program.

Some of its main functions are explained below.

Variables

Variables can be used for storing values.

E.g. int a=10;

Functions

Functions can be used to peform operations.

E.g. public int power(n, e) calculates and returns the result of n^e.

Objects

Objects can contain variables and functions.

E.g. A person object can contain the variables name and date of birth with a function to calculate their age.
